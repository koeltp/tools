# 全能工具箱

一个集成了多种实用工具的开发工具箱，完全在浏览器中运行，无需任何外部依赖。

## 功能特性

1. **时间戳转换工具**
   - 时间戳与日期格式相互转换
   - 支持多种日期格式显示
   - 实时显示当前时间和时间戳

2. **JSON工具**
   - JSON格式化与压缩
   - JSON语法验证
   - JSON字符串转义与反转义

3. **MD5加密工具**
   - 生成文本的MD5哈希值
   - 用于数据校验和简单加密

4. **AES加密/解密工具**
   - 模拟AES加密解密功能
   - 支持16/24/32字节密钥
   - 加密/解密模式切换

5. **编码/解码工具**
   - Base64编码/解码
   - URL编码/解码

6. **密码生成器**
   - 自定义密码长度和字符类型
   - 实时密码强度评估
   - 生成多个密码选项
   - 一键复制到剪贴板

## 使用方法

1. 将所有文件下载到同一目录
2. 使用现代浏览器打开 `index.html`
3. 通过左侧导航栏切换不同工具
4. 根据需要输入数据并点击相应按钮处理

## 文件结构
```
- `index.html` - 主HTML文件，包含所有工具的界面
- `styles.css` - 样式文件，定义页面布局和样式
- `scripts/` - 包含各个工具的JavaScript实现
  - `timestamp.js` - 时间戳转换工具
  - `jsonTool.js` - JSON工具
  - `md5.js` - MD5加密工具
  - `aes.js` - AES加密/解密工具
  - `encoding.js` - 编码/解码工具
  - `passwordGenerator.js` - 密码生成器
```
